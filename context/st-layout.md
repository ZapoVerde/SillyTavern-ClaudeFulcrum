# ST Layout

Standard SillyTavern directory structure relative to `/home/node/app`.

| Directory | Contents |
|-----------|----------|
| `data/default-user/characters/` | Character cards — PNG with embedded JSON, or bare JSON |
| `data/default-user/chats/` | Chat logs, one subdirectory per character |
| `data/default-user/worlds/` | Lorebooks / World Info JSON files |
| `data/default-user/presets/` | Text generation presets (textgenerationwebui, openai, etc.) |
| `data/default-user/context/` | Context template files |
| `data/default-user/themes/` | UI theme JSON files |
| `data/default-user/user/images/` | User-uploaded and generated images |
| `data/default-user/user/files/` | Extension output files |
| `public/scripts/extensions/third-party/` | Third-party extensions |
| `plugins/` | Server-side ST plugins |
| `src/` | ST server source code |
| `public/` | ST frontend static files |
